Guide to develop with MEAN (MySQL Express AngularJS and NodeJS)

Guide for Sample Registration page

important ones:

1 package.json → it’s like Maven for NodeJS. It gives information to node runtime (like the app name or the main .js to execute) and to npm (to download all the dependencies needed by node runtime)

2.node_modules → npm will download all the app dependencies on this folder. This folder should never be in your code repo so yeoman includes it into .gitignore

3.** mai-server.js** → Server file it contains script for Nodejs , Include libraries and set port number also set routers, database configurations.
syntax for run this file
node main-server.js

followed by

Step1: Git install
Git clone https://github.com/suneel2you/NodeJS-with-AngularJS.git master(branch)

Step2: install node
sudo apt-get install node

Step3: Install npm files
sudo npm install ---dependencies--

Step4: It contain database node_angular.sql file, Import this file to local system.
set configurations in dbconf.js file.

Step5: run the main file
node main-server.js
